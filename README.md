Nombre del proyecto:
Conversor de Monedas en C

Descripción:
Este proyecto consiste en una aplicación desarrollada en lenguaje C que permite convertir entre quetzales, dólares estadounidenses y euros. El programa solicita al usuario la moneda de origen, la moneda de destino y el monto a convertir, mostrando el resultado en pantalla.

Prerrequisitos (tecnologías a instalar):
Compilador de C (GCC)
Visual Studio Code u otro editor de código
Git (opcional, para clonar el repositorio)

Pasos para su ejecución:
Clonar o descargar el repositorio en la computadora.
Abrir la carpeta del proyecto en un editor de código.
Compilar el programa usando un compilador de C.
Ejecutar el programa desde la terminal.
Ingresar los datos solicitados para realizar la conversión de monedas.
